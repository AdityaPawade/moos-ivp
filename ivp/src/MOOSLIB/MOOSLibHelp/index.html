MOOSLib Documentation

What MOOS and MOOSLib?

MOOS is a Minimal Odyssey Operating System. It is built around a client server architecture designed to deliver an efficient small footprint sytem running on any platform. A fundamental tennant of the design is that it should be easy to develop/add new functionality to the vehicle without needing expertise in every part of the system. This documentation describes the communications library that transparently binds the MOOS components together into a whole.
